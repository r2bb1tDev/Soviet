# UI и дизайн

## Философия дизайна

Soviet Messenger использует **терминальный / MS-DOS эстетику** — зелёный фосфорный монитор, моноширинный шрифт, пиксельные границы без скруглений. Интерфейс вдохновлён ICQ и CRT-терминалами 80-х, адаптирован для современных ОС.

Принципы:
- **Терминальная эстетика** — тёмный фон (`#0A0A0A`), зелёный акцент (`#2ECC71`), мятный текст (`#C8FFD4`)
- **Функция важнее красоты** — но красота не противоречит функции
- **Минимализм** — только необходимые элементы, никакой рекламы
- **Единый стиль** — нет скруглений, пиксельные границы, шрифт `Share Tech Mono` / `VT323`
- **CRT-эффект** — scanline overlay через CSS `::after` pseudo-element

## Логотип и иконки

### Белый медведь (основной логотип)

Логотип — **полярный белый медведь**, смотрящий на зрителя, слегка наклонив голову.

**Стиль:**
- Минималистичный флэт-дизайн
- Мягкие округлые линии
- Не мультяшный, не photorealistic
- Значимые элементы: глаза, уши, лапы

**Файлы:**
```
assets/logo/
├── bear_full.svg           # Логотип + текст "Soviet"
├── bear_icon.svg           # Только медведь (для иконки приложения)
├── bear_icon_dark.svg      # Версия для тёмного фона
└── bear_icon_light.svg     # Версия для светлого фона
```

### Системный трей

**Состояние 1: Нет новых сообщений**
- Иконка: белый медведь (монохромная)
- Статичная, без анимации
- Автоматически адаптируется к цвету панели (светлый/тёмный)

**Состояние 2: Есть непрочитанные сообщения**
- Иконка: белый медведь + маленький конвертик (в лапе или рядом)
- Может мигать (animation) 3-5 раз для привлечения внимания
- Мигание прекращается через 10 секунд
- Иконка с конвертом остаётся до прочтения сообщений

**Состояние 3: Доступно обновление (с v2.2)**
- OS-уведомление при обнаружении новой версии
- В трее появляется пункт "⬆ Доступно обновление vX.X.X"

**Контекстное меню (правый клик на трее):**

## Темы и адаптация

### Автоматическое определение темы ОС

**Windows:**
```
Registry: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Themes\Personalize
AppsUseLightTheme: REG_DWORD
  0 = тёмная тема
  1 = светлая тема
```

**macOS:**
```
defaults read -g AppleInterfaceStyle
  nil / отсутствует → светлая тема
  Dark → тёмная тема
```

**Linux (GNOME):**
```
gsettings get org.gnome.desktop.interface gtk-application-prefer-dark-theme
  true → тёмная тема
  false → светлая тема
```

**Linux (KDE):**
```
~/.config/kdeglobals [Colors:Window]
BackgroundColor → анализируем яркость
```

Приложение подписывается на **системные события смены темы** и переключается **без перезагрузки**.

### CSS-переменные (React)

Палитра — терминальный зелёный фосфор (с v2.0, смягчён в v2.0 для читаемости):

```css
[data-theme="dark"] {
  /* Фоны */
  --bg-primary:    #0A0A0A;   /* почти чёрный */
  --bg-secondary:  #0F1A0F;   /* тёмный хвойный */
  --bg-tertiary:   #0D150D;
  --bubble-out-bg: #071A0F;   /* свои сообщения */

  /* Текст */
  --text-primary:   #C8FFD4;  /* мягкий мятный */
  --text-secondary: #6ECC8A;  /* средний зелёный */
  --text-muted:     #2E6642;  /* приглушённый */

  /* Акцент */
  --accent:       #2ECC71;    /* изумрудный */
  --accent-hover: #27AE60;

  /* Границы и тени */
  --border:  rgba(46,204,113,0.15);
  --glow:    0 0 7px rgba(46,204,113,0.35);
  --shadow:  rgba(46,204,113,0.07);

  /* Статусы */
  --online:     #2ECC71;
  --away:       #F1C40F;
  --busy:       #E74C3C;
  --offline:    #555;

  /* Сообщения */
  --input-border: rgba(46,204,113,0.3);
  --header-text:  #C8FFD4;
  --row-unread-badge: #2ECC71;
}

[data-theme="light"] {
  --bg-primary:   #F0FFF4;
  --bg-secondary: #E6F9EE;
  --text-primary: #1A3A1A;
  --accent:       #27AE60;
  --border:       rgba(46,204,113,0.25);
}
```

### Ручное переключение темы

В **Настройки** → **Оформление**:
```
Тема:
○ Светлая
○ Тёмная
○ ● Автоматическая (следует ОС)
```

## Звуки

| Событие | Звук | Длительность |
|---------|------|------------|
| Входящее сообщение | ICQ-звон (модифицированный) | 1 сек |
| Исходящее сообщение | Тихий щелчок | 300 мс |
| Пользователь вошёл | Короткий звон | 500 мс |
| Пользователь вышел | Приглушённый звук | 400 мс |
| Новый запрос контакта | Уведомительный сигнал | 600 мс |

**Управление:**
- Все звуки можно отключить в Настройки → Уведомления
- Громкость управляется громкостью системы
- Звуки НЕ воспроизводятся при режиме "Do Not Disturb" ОС

## Доступность (Accessibility)

### Навигация с клавиатуры

- `Tab` — переход между элементами
- `Shift+Tab` — переход в обратном направлении
- `Enter` — активация кнопки/ссылки
- `Space` — активация кнопки (alt)
- `Escape` — закрытие диалога
- `Ctrl+F` — поиск в текущем окне
- `Ctrl+Enter` — отправка сообщения

### Программы чтения с экрана

- ARIA-метки для всех интерактивных элементов
- `role="button"`, `role="link"`, `role="list"`, etc.
- `aria-label` для иконок без текста
- `aria-live="polite"` для уведомлений

### Масштабирование

```
Настройки → Оформление → Масштаб:
○ 100% (по умолчанию)
○ 125%
○ 150%
○ 200%
```

### Высокий контраст

При включённом режиме "Высокий контраст" ОС:
- Цвета усилены (более насыщенные)
- Белые линии на тёмном фоне (или наоборот)
- Отключаются некоторые анимации
- Увеличивается размер фокуса элемента

## Анимации

**Принцип:** анимации **быстрые и ненавязчивые**. Ни одна анимация не должна мешать работе.

| Элемент | Анимация | Длительность |
|---------|----------|---|
| Появление нового сообщения | Fade in + slide up | 150 мс |
| Изменение статуса контакта | Color change + scale | 200 мс |
| Открытие чата | Slide from left | 120 мс |
| Иконка в трее (новые сообщения) | Pulse / Blink | 300 мс (3x) |
| Кнопка при наведении | Brightness change | 100 мс |

**Отключение анимаций:**
- При включённом системном параметре "Reduce Motion" (prefers-reduced-motion)
- Все анимации заменяются на instant transitions

## Цветовая слепота

Цвета используют не только для кодирования информации:

```
❌ Только цвет:
   [🔴 Error] [🟡 Warning] [🟢 Success]

✓ Цвет + иконка:
   [❌ Error] [⚠️ Warning] [✓ Success]
```

Статусы пользователей:
```
🟢 Online  + слово "Online"
🟡 Away    + слово "Away"
🔴 Busy    + слово "Busy"
⚪ Offline + слово "Offline"
```
